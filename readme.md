# 🛒 Shopee Cart Simulator

Simulação de um carrinho de compras utilizando JavaScript puro (Node.js), com funcionalidades básicas como adicionar itens, remover, deletar e calcular o total.

---

## 🚀 Funcionalidades

- Adicionar item ao carrinho
- Remover unidade de um item
- Deletar item completamente
- Calcular total do carrinho
- Listar itens do carrinho

---

## 🧠 Conceitos utilizados

- Manipulação de arrays (`push`, `splice`, `findIndex`, `reduce`)
- Funções assíncronas (`async/await`)
- Organização por módulos (ES Modules)
- Encapsulamento de lógica (services)
